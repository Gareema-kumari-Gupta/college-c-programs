## ▶️ How to Run

1. Install a C compiler (such as GCC or Code::Blocks)

2. Compile the program:

```bash
gcc filename.c -o output
```

3. Run the program:

```bash
./output
```

Example:

```bash
gcc Bubble_sorting.c -o bubble
./bubble
```
